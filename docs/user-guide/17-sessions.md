# Temple Recollections (Sessions)

Tehuti preserves every exchange as a Temple Recollection.

## Nature

A Recollection holds the full history, tool verdicts, memory entries, and state for continuity.

## Beginning and Resuming

- New Recollection: `tehuti` or `/begin`
- Resume: `/recall` or `tehuti --recall <id>`
- Continue latest: `tehuti -c`

Recollections are stored in `~/.tehuti/recollections/`.

## Maintenance

- `/compact` to condense the papyrus
- `/restore` to return to an earlier state
- Auto-compaction when the scroll grows long

These mechanisms ensure enduring truth.
