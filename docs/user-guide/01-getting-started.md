# Getting Started with Tehuti

Tehuti is the self-evolving agent harness, the Architect of Truth.

## Invocation

To begin an audience with Tehuti:

```bash
tehuti
```

Or with an initial inscription:

```bash
tehuti --inscribe "establish the foundation of the new core"
```

## The Chamber

The interface presents the Living Papyrus (conversation history) and the Quill (input).

Inscriptions starting with `/` invoke the sacred rites of the Scribe.

## Core Principles

- Truth above speed
- Balance (Ma'at) in all actions
- Enduring knowledge through the Living Archive
- User extensions only — no hard-coded personal tools

For full capabilities, consult the other records in this guide.
