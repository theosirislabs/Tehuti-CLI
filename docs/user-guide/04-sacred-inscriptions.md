# Sacred Inscriptions

The petitioner may inscribe commands beginning with `/` to direct the Architect.

## Core Inscriptions

- `/begin` — Open a new Recollection
- `/recall` — Resume a prior Recollection
- `/compact` — Condense the current scroll
- `/restore` — Return the chamber to an earlier state
- `/records` — List active Recollections
- `/help` — Consult the sacred records

## Interface Rites

- `/temple-info` — Reveal the current state
- `/settings` — Adjust the configuration of the Temple

Skills and extensions may manifest as additional Inscriptions when properly marked.

The full list is available through the inscription menu.
